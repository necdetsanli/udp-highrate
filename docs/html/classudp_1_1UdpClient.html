Detailed Description

High-rate UDP client using a pluggable ISocket.

Lifecycle:

  • Construct with a concrete ISocket and ClientConfig.
  • Call start() to spawn the worker thread and begin paced sending.
  • Optionally call join() to block until completion.
  • Call stop() for an early, graceful shutdown.

Behavior:

  • The worker run_loop() constructs datagrams (optionally including a

    PacketHeader set by the caller in the source buffer) and transmits them

    at the configured PPS. It updates Stats on the hot path using relaxed atomics.

  • When available, the socket implementation batches sends to minimize syscalls.
  • The loop exits when the configured duration elapses or when stop() is called.
