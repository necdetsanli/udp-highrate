Detailed Description

Background HTTP endpoint that serves metrics derived from udp::Stats.

  • start spawns a thread that listens on port_ and serves requests.
  • stop requests a graceful shutdown and joins the thread.
  • render builds the plaintext payload from the current stats snapshot.

The implementation is purposely minimal (no TLS, no routing beyond a single endpoint such as /metrics) to keep the hot path free of heavy dependencies.
