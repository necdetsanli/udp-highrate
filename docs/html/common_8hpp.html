Detailed Description

Common wire-level types and tiny utilities for the UDP high-rate demo.

This header defines the on-the-wire packet header used by both client and server,

plus two helpers:

  • a monotonic nanosecond timestamp provider (udp::now_ns),
  • and a human-readable rate formatter (udp::human_rate).
Note
All functions here are thread-safe and lock-free.
Warning
The wire layout does not perform endian conversion; see notes on udp::PacketHeader.
