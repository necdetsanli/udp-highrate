Detailed Description

Runtime configuration for UdpClient.

  • server_ip : Dotted IPv4 address of the server (e.g., "127.0.0.1").
  • port : Server UDP port (host byte order).
  • pps : Target packets-per-second to transmit (global rate).
  • seconds : Total run duration; the client stops after this many seconds.
  • payload : Payload size per datagram (bytes), not including any header added by caller.
  • batch : Preferred number of messages per send syscall (throughput vs latency trade-off).
  • id : Optional client identifier (for logging or payload tagging).
  • verbose : If true, prints periodic rate/counter lines to stdout.
