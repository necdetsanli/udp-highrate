Detailed Description

Runtime configuration knobs for UdpServer.

  • port : UDP port to bind on the local host (host byte order).

  • batch : Target number of datagrams to process per syscall. Larger values

    reduce syscall overhead (PPSâ†‘) at the cost of slightly higher per-packet latency.

  • echo : If true, mirror incoming datagrams back to their source address.
  • reuseport : If true, attempt to enable SO_REUSEPORT (multi-worker setups).
  • verbose : If true, log periodic rate/counter lines to stdout.
  • metrics_port : TCP port for /metrics endpoint (0 disables HTTP metrics).
