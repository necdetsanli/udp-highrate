Detailed Description

Minimal HTTP server that exposes runtime metrics for scraping.

This helper runs a tiny HTTP server on a background thread and renders the current counters from udp::Stats in a text format suitable for simple scraping (e.g., Prometheus exposition format). It is intentionally small and dependency-free so it can be embedded directly into tests and demos.

Usage
udp::Stats stats;
udp::MetricsHttpServer http(stats, 9100);
http.start();
// ... run workload, then scrape http://<host>:9100/metrics ...
http.stop(); // or rely on destructor to stop and join
